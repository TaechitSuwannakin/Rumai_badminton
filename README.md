Project Rumai-badminton นี้เป็นเพียงโปรเจค Demo ที่สามารถใช้ได้จริง แต่อาจจะมีข้อมูลที่ยังไม่ครบถ้วน หรือ ถูกต้องทุกประการจึงขออภัยในส่วนนี้ด้วยนะครับ 
ในส่วนของเนื้อหาและที่มาของโปรเจคนี้ โดย เริ่มามาจากความชอบส่วนตัวสำหรับกีฬาแบดมินตัน และเกิดข้อสงสัยที่เกี่ยวกับการเลือกไม้ของผู้เล่นในระดับมือใหม่หรือมือระดับปานกลาง 
จึงทำให้ผมได้ พัฒนาเว็บไซต์นี้ขึ้นมา  ซึ่งมีฟังก์ชั่นดังนี้ 
1. สามารถเข้าร่วมก๊วนไลน์และเมื่อกดแผนที่จะสามารถเข้า Google map ไปยังสถานที่จัดก๊วนได้เลย
2. มีตัวเลือก ความชอบของสไตล์การเล่น , ฟิลลิ่งหัวไม้ที่ชอบ , ราคางบประมาณ เพื่อนำไปประมวลผลค้นหาไม้ให้ตรงกับที่เลือกมา
3. สามารถกดรีวิวให้ดาว และคอมเม้นต์ได้ 
4. สามารถกดดูราคาสินค้าได้ จะทำการลิงค์ไปยังหน้าค้นหาของ shopee
